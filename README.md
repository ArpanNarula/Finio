# Finio - Personal UI/UX Case Study

This is my end-to-end product design project for a personal finance app built for young Indian professionals.

Live project: https://finio-gamma.vercel.app/

## Why I built this

I wanted to solve a real behavior problem, not just make pretty screens:
people earning decent money still struggle to understand where it goes.

So I designed Finio as a complete product experience:
- clear money tracking
- budget visibility
- goal-based saving
- AI guidance that tells users what to do next

## What I shipped

This repo includes a full portfolio package, not just mockups.

- `case-study/index.html` - full case study narrative (problem, process, decisions, outcomes)
- `prototype/index.html` - interactive mobile prototype with 7 flows
- `design-system/index.html` - visual system (tokens, typography, spacing, components)
- `wireframes/index.html` - low-fi screens + sitemap + user flow
- `research/index.html` - personas, pain points, competitor breakdown, journey map
- `index.html` - landing hub for all sections

## Design direction

- Base theme: premium dark interface for focus and trust
- Accent: green used for positive financial actions and progress
- Type system: Syne for high-emphasis headings, DM Sans for readable body text
- Interaction style: minimal friction, quick scanability, high contrast hierarchy

## Run locally

No build setup needed.

1. Clone or download this repo
2. Open `index.html` in a browser
3. Navigate to each section from the landing page
